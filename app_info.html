Self-Healing

LauncherBeta v3.0 introduces advanced observers that monitor the system state. If a package is uninstalled, the launcher automatically purges dead shortcuts and refreshes the database without user intervention.

Async Loading

Using Kotlin Coroutines and Coil, every icon and wallpaper is loaded in the background. This prevents UI thread jank, ensuring smooth scrolling even with thousands of installed applications.

Android 16 Ready

We've implemented full Edge-to-Edge enforcement, including handling of display cutouts and navigation bar paddings, ensuring LauncherBeta looks native on the latest Android hardware.
