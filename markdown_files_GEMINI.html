EchoCustom Pro - Project Context (v3.0+)

🌟 Overview

EchoCustom Pro is a high-performance Android personalization suite. Originally a notification sound manager, it has evolved into a comprehensive launcher and utility suite focused on accessibility, modularity, and user-centric design.

Core Pillars:

  • Modular Dashboard: A highly customizable home screen with a "Maid" widget interaction system.
  • Sound Mapping: Per-app and per-contact audio overrides using NotificationListenerService.
  • Elder Mode: A specialized UI branch providing high-visibility app grids.

🏗 Project Structure

📁 Top-Level Organization
