Modular Architecture

High-Performance Inference Engines

Project You v2 features a modular engine system, allowing you to choose the best model for your device's capabilities.

  • MediaPipe GenAI: Powering the next generation of SmolLM2 models for mobile-first efficiency.
  • llama.cpp (JNI): Native support for GGUF-based models, including Llama 3 and Mistral architectures.
  • Assistant Service: A persistent Android Foreground Service ensuring the assistant is ready when you need it.
  • Model Manager: Intelligent local storage management for downloading and swapping model files.
Developer Utilities

AI Model Management Tools

Project You includes specialized Python scripts to help developers and power users manage their local model libraries.

Python Utility Scripts

  • convert.py: Transform raw .safetensors checkpoints into MediaPipe-compatible .task files.
  • list_tensors.py: Deep inspection of model architecture and tensor weights for debugging and optimization.

Core Implementation Files

- YouAiEngine.kt: SmolLM2/MediaPipe orchestration. - LlamaEngine.kt: Native JNI wrapper for LLM inference. - AssistantService.kt: Service lifecycle and wake-word logic. - ModelManager.kt: Local storage and .task file management.
Setup Guide

Getting Started

To experience Project You v2, you'll need an Android device running API 26 or higher and a compatible model file.

  1. Download the v2 Release APK from the downloads page.
  2. Obtain a SmolLM2 .task file (see conversion scripts above).
  3. Place the model in files/models/smollm2.task within the app's internal storage.
  4. Launch the app and experience the future of local AI.
