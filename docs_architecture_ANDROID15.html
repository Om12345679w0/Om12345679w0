Android 15 Deployment Configuration Guide

Overview

This document outlines all configuration updates made to support Android 15 (API 35) deployment across all Android mobile phones.

1. Configuration Files Updated

1.1 AndroidManifest.xml

  • Added POST_NOTIFICATIONS permission.
  • Added READ_MEDIA_AUDIO permission.

1.2 app/build.gradle.kts

compileSdk = 35
minSdk = 26
targetSdk = 35

2. Device Compatibility

Supported on CPU architectures ARM64, ARM, x86, x86_64. Min RAM 256MB.
