System Capabilities

EchoCustom Pro is a high-performance Android utility that leverages local machine learning and voice recognition to create a responsive, intelligent system assistant.

🎙️ Voice Intelligence (v4.1 Beta)

  • Maid Voice Assistant: On-device speech recognition allowing for hands-free system commands and interactions.
  • Customizable Wake Word: Summon the assistant with "Echo" or configure your own unique wake word in the settings.
  • Manual Trigger Button: A dedicated UI trigger located directly above the pinned apps row for instant activation.
  • Note: These features are currently in Beta and undergoing active stability testing.

🧠 ML-Powered Intelligence (v4.0)

  • Language Identification: Real-time detection of notification languages (EN, ES, FR) to apply appropriate sentiment filters.
  • Entity Extraction: Automatically identifies dates, phone numbers, and addresses within notifications to provide actionable hints.
  • Intelligent Triage: Classification of alerts into SOCIAL, WORK, and SYSTEM categories with multi-language sentiment analysis.

🎨 Dynamic UI Widgets

  • Random SVG Icon: A new 1x2 vertical widget that randomly selects and renders vector icons from a library of 1000+ assets.
  • Instant Refresh: Tap-to-cycle logic using custom broadcast intents for immediate visual feedback.
  • Vector Fidelity: High-resolution rendering at any scale using the AndroidSVG engine.

🛡️ System & Accessibility

  • Shake-to-SOS: Accelerometer-based gesture to trigger emergency calls or notifications directly from the launcher.
  • SAO Resource HUD: Real-time monitoring of CPU waves, RAM, and network throughput with a high-fidelity interface.
  • Service Healing: Background monitors that automatically restore the Notification Listener if killed by the OS.
  • Accessibility Gestures: Double-tap to lock, custom draw-over gestures, and high-visibility Elder Mode support.
