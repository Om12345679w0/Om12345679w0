Privacy Policy

LAST UPDATED: JUNE 25, 2026

Your privacy is non-negotiable. LauncherBeta is built from the ground up with a local-first architecture. We believe that your device should be your private sanctuary.

1. No Data Collection

LauncherBeta does not collect, store, or transmit any personal data, usage statistics, or analytics to our servers or any third parties. Everything stays on your device.

2. On-Device Processing

  • App Indexing: All app search and sorting logic happens locally.
  • Icon Rendering: Third-party icon packs and procedural fallbacks are processed on-device.
  • Web Suggestions: Central Hub search queries are sent to DuckDuckGo's API only when you type in the search bar. This is a direct request and we do not intercept or log it.

3. Permissions

We only ask for the minimum required permissions to function as a launcher:

  • Query All Packages: To show your installed apps in the drawer.
  • Storage/Media: To allow you to set custom video wallpapers and access icon packs.
  • Biometric: Used exclusively to unlock the "Hidden Apps" folder (processed via Android System Biometric API).

4. Third-Party Services

LauncherBeta contains zero third-party tracking, advertising SDKs, or analytics packages. We use standard Android libraries for core functionality (Jetpack, Coil, Room).

5. Security

Your launcher configuration and local database are stored in the app's private sandbox, protected by the Android operating system's security layers.
