WidgetQuote

Inspirational quotes, beautifully delivered to your home screen.

Project Overview

Inspiration on Demand

WidgetQuote is a modern Android application designed to keep you inspired throughout the day. Using the latest Android technologies, it provides a seamless experience for managing and displaying your favorite quotes right on your home screen.

✨ Key Features

  • Home Screen Widget: Periodically refreshed quotes using Jetpack Glance for a modern, responsive feel.
  • Quote Management: Easily add, edit, or delete quotes from your personal collection.
  • Import & Export: Full backup support via JSON files, making it easy to share or move your collection.
  • Customizable Refresh: Control how often your widget updates, from 15 minutes to 24 hours.
  • Offline-First: Powered by a local Room database for instant access without an internet connection.

Technical Stack

Built with Modern Android

WidgetQuote serves as a showcase for modern Android development practices and libraries.

  • Jetpack Compose: For a fully declarative and responsive user interface.
  • Glance: Bringing the power of Compose to home screen widgets.
  • Room Database: Robust local data persistence and management.
  • WorkManager: Reliable background processing for widget refreshes.
  • DataStore: Modern, type-safe preference storage for app settings.
  • MVVM Architecture: Clean separation of concerns for maintainability.
Data Portability

JSON Import/Export

Your quotes belong to you. WidgetQuote makes it easy to backup your collection or import new ones using a simple JSON format.

Expected JSON Structure

[ { "text": "The only way to do great work is to love what you do.", "author": "Steve Jobs" }, { "text": "Innovation distinguishes between a leader and a follower.", "author": "Steve Jobs" } ]

The importer is resilient and will automatically handle extra fields like id or isCustom if present in your backup files.
